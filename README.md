# Business Website

## Project Overview
This is a **responsive business website** built using **React.js**.  
It features a modern, mobile-friendly design, reusable components, and showcases company services, portfolio, products, and contact functionality.  

---

## Features
- Responsive layout for desktop, tablet, and mobile  
- Pages: Home, About, Services, Portfolio, Products, Cart, Contact  
- Reusable components: Navbar, Footer, ProductCard  
- Modern CSS styling and animations  
- Easy to extend and maintain  

---

## Installation & Setup

1. **Clone the repository**

```bash
git clone https://github.com/SGauri24/business-website.git

Navigate to the project folder

cd business-website


Install dependencies

npm install


Run the application locally

npm start

The website will run at: http://localhost:3000

Build for Production
npm run build


Compiles the app into the build/ folder ready for deployment

Notes

node_modules/ and build/ are ignored by Git

Optional folders (docs/, lighthouse/, tests/, .github/) can be added for extra documentation or testing

Components are structured for scalability and reusability
